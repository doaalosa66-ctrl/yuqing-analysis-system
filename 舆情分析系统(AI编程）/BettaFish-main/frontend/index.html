    使用指引
    100%
    01
    配置 LLM 引擎
    点击搜索框右侧参数图标，展开高级面板，选择「LLM 配置」填写 API Key 并保存。首次使用必须完成此步骤。
    前置条件
    02
    输入舆情主题
    在中央搜索框输入你想分析的品牌、事件或关键词，例如「某品牌危机公关」。支持中英文混合输入。
    核心操作
    03
    选择数据源与深度
    展开参数面板，选择「MindSpider 公开抓取」或「私有数据库」，并设置抓取深度。深度越高，分析越全面，耗时也越长。
    可选配置
    04
    等待 Agent 协作分析
    提交后，Query / Media / Insight 三个智能体将并行工作。你可以在「系统思考流」终端实时观察每个 Agent 的进度与发现。
    自动执行
    05
    生成与导出报告
    三个 Agent 完成后，切换到「报告引擎」Tab，点击「生成最终报告」。报告生成后可通过右下角 FAB 按钮导出 PDF / HTML / Markdown。
    输出结果
    可上传自定义 .md 模板文件，让报告结构完全符合你的业务需求。
